Habilidades

En esta sección, presento un conjunto de habilidades que he desarrollado a lo largo de mi trayectoria como estudiante de programación

HTML

Lenguaje de marcado utilizado para estructurar el contenido en la web. Permite definir elementos como encabezados, párrafos, enlaces y otros componentes de una página.

CSS

Hojas de estilo en cascada que se usan para describir la presentación y el diseño de documentos HTML. Permite ajustar colores, fuentes, márgenes y otros aspectos visuales.

BOOSTRAP

Framework de código abierto para desarrollar sitios web y aplicaciones móviles. Ofrece una colección de herramientas y componentes predefinidos que facilitan la creación de interfaces responsivas.

PYTHON

Lenguaje de programación de alto nivel conocido por su sintaxis clara y su versatilidad. Es ampliamente utilizado en desarrollo web, análisis de datos, inteligencia artificial y automatización.
